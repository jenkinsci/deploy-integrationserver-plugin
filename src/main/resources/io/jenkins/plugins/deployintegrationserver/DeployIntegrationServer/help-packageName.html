Integration Server/ Microservices Runtime package name regular expression or name of packages comma-separated that needs to be uploaded and/or installed. Examples
  • CustomPackage.zip - A specific package name.
  • [a-zA-Z]*.zip - All packages with letters in their name.
  • [a-zA-Z0-9]*.zip - All packages with letters and numbers in their names.
  • AH[a-zA-Z0-9]*.zip - All packages having 'AH' as prefix followed by letters and numbers in their names.
